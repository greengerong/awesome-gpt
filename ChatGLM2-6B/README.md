# ChatGLM2-6B

## setup

```shell

    git clone https://github.com/THUDM/ChatGLM2-6B
    cd ChatGLM2-6B
    pip install -r requirements.txt

```

* transformers 库版本推荐为 4.30.2，torch 推荐使用 2.0 及以上的版本，以获得最佳的推理性能

### cli demo

```shell

   python cli-demo.py

```

### web demo

```shell

   python web_demo.py

```
